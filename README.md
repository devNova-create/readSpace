# ReadSpace - PDF Viewer App

ReadSpace is a modern Android PDF Viewer application designed to provide a smooth and comfortable document reading experience. The app offers multiple reading modes, bookmarking functionality, and a clean user-friendly interface for seamless PDF navigation.

## Features

* Open and view PDF files
* Smooth PDF rendering
* Dark mode support
* Bookmark pages for quick access
* Horizontal scrolling book mode
* Zoom in and zoom out functionality
* Fast and responsive UI
* Simple and modern design

## Technologies Used

* Kotlin
* Android Studio
* Android SDK
* XML
* PDF Viewer Library

```bash id="6c5v3v"

```

Sync Gradle files and run the application on an emulator or physical Android device.

## How to Run

1. Open the project in Android Studio
2. Connect an Android device or start an emulator
3. Click the Run button
4. Install and launch the app
